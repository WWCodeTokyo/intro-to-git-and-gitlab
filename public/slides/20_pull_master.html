We have one last thing to do! We just merged our changes from our branch into the master branch on the server. Now we need to update our local copy of 'Master' to include these changes.

Let's check out the master branch using the 'git checkout' command.

Use 'git pull' to update your local copy of the master branch. This command incorporates changes from a remote repository into the current branch.

Run 'git status' one last time to make sure you are all up to date.
