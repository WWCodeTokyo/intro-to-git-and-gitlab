Setup Account with GitLab

https://gitlab.com
Navigate to gitlab.com and select the 'Register' tab. You will need to enter details and confirm your email address. Then you will be able to log-in using the credentials with which you registered.

**If they already have a GitHub or bitbucket account, let them know they can sign-in using that account and import projects.

First look at GitLab

Once logged in, you can do a number of things.

You can create a project. A project is where you house your files (the repo), plan your work (using GitLab issues), and publish documents (GitLab wiki).

You can have a project without any code and use issues to track and plan work or the wiki to publish documents.

You can have a repo with your codebase and never use issues or the wiki.

It is all up to you!

You can create a group. Groups are the best way to manage multiple projects and members.

You can also explore public projects or learn more about GitLab using our documentation.
