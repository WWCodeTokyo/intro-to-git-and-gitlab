Uh-oh! We got an error, but why?

We created the branch in our local working repository. Your local branches aren’t automatically synchronized to the remotes server — you have to push the branches you want to share explicitly.
Set Origin
      
          $ git push --set-upstream origin [BRANCHNAME]
      
    
      
          Branch [BRANCHNAME] set up to track remote branch test-branch from origin
Luckily, Git gives you the command needed to do this. Use the 'set origin' command to push this local branch to the remote server.

You should see a flurry of activity ending with the following message: "Branch [BRANCHNAME] set up to track remote branch test-branch from the origin."
